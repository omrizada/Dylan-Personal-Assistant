# Activity Digest

Weekly digest of brain updates, decisions, and learnings.

## Instructions

1. Read `brain/learnings/session-notes.md` for recent session context
2. Read `brain/context/decisions-log.md` for recent decisions
3. Read `brain/learnings/patterns.md` for new patterns
4. Read `brain/learnings/feedback.md` for recent corrections
5. Read `brain/context/projects.md` for status changes
6. Compare last_updated dates across all brain files to identify changes

## Output Format

```
## Weekly Digest: [date range]

### Key Decisions
- [Decision 1] — [rationale]

### Brain Updates
- [X] files updated this week
- Notable changes: [list]

### Patterns Observed
- [Pattern 1]

### Open Items
- [Follow-up 1]
- [Follow-up 2]

### Stale Areas
- [Files not updated in 2+ weeks]
```
