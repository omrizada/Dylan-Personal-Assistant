Mac package manager. Required for installing Node.js and Python.

What is a terminal?

A terminal is a text-based way to interact with your Mac. To open it:

  1. Press Cmd + Space to open Spotlight
  2. Type Terminal
  3. Press Enter

You'll see a window with a blinking cursor. This is where you'll paste commands.

Step 1: Install Xcode Command Line Tools

xcode-select --install

A dialog will pop up. Click Install, then Agree. Wait 5-15 minutes for it to finish.

Step 2: Install Homebrew

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

You'll be asked for your Mac password. When you type it, nothing appears on screen -- this is normal! Just type your password and press Enter.

Step 3: Add Homebrew to your PATH

Not sure which Mac you have? Click the Apple menu at top-left, then About This Mac. If it says M1, M2, M3, or M4, you have Apple Silicon.

Apple Silicon (M1/M2/M3/M4):

echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zprofile && eval "$(/opt/homebrew/bin/brew shellenv)"

Intel Mac:

echo 'eval "$(/usr/local/bin/brew shellenv)"' >> ~/.zprofile && eval "$(/usr/local/bin/brew shellenv)"
Common errors

"Permission denied" -- Try running with sudo or check that you own the /usr/local directory.

"Command not found: brew" -- You need to add Homebrew to your PATH (Step 3 above). Try closing and reopening your terminal.

"Already installed" -- That's fine! Run brew update to get the latest version.

Set Up Telegram

Telegram lets you message Dylan from your phone, like texting a friend. You can skip this and use the terminal instead.

Connect Telegram to message Dylan from anywhere. Optional but recommended.

Optional Telegram integration.

Connect via Telegram (recommended)

A Create a bot

Open Telegram and search for @BotFather

  1. Send /newbot
  2. BotFather asks: "How are we going to call it?" -- type a name like Dylan Assistant
  3. BotFather asks for a username -- type something ending in bot, like dylan_helper_bot
  4. BotFather gives you a token. It looks like: 1234567890:ABCdef...

You got this from BotFather in the previous step.

Treat this token like a password. Don't share it publicly.

B Turn off Group Privacy

By default, bots can't read messages in groups. We need to change this so Dylan can see your messages.

  1. In your BotFather chat, send /setprivacy
  2. Select your bot
  3. Tap Disable
  4. BotFather confirms: "Success! The new status is: DISABLED."
Important: If you skip this step, Dylan won't be able to read your messages in the group.

C Create a Forum group

  1. In Telegram, create a new Group
  2. Tap the pencil/compose icon, then New Group
  3. Name it (e.g., "Dylan HQ")
  4. Go to Group Settings, then Edit, then enable Topics

Create these 5 topics:

  • 💼 Work
  • 🚀 Side Project
  • 🏠 Personal
  • 💬 General
  • 🛠 Development

D Add bot and make admin

  1. Add your bot to the group (search for its username)
  2. Go to Group Info, then Administrators, then add your bot as admin

The bot needs admin rights to send messages in topics.

E Get your IDs

  1. Add @RawDataBot to the group temporarily
  2. It posts a message with the group's chat_id (a negative number like -1001234567890)
  3. Now remove @RawDataBot from the group
  4. Send a private message to @userinfobot to get your user ID

F Get topic thread IDs

In Telegram Desktop, right-click a message in each topic and select Copy Message Link.

The link format is: https://t.me/c/GROUP_ID/THREAD_ID/MSG_ID

The middle number is the thread_id.

If you created the topics in the order listed above, the defaults should be correct.

G Install the Telegram plugin

In Claude Code, run:

/plugin install telegram@claude-plugins-official

If the marketplace is not added yet:

/plugin marketplace add anthropics/claude-plugins-official

Configure with your token:

/telegram:configure YOUR_TOKEN

Exit Claude Code and relaunch with:

claude --channels plugin:telegram@claude-plugins-official

DM your bot on Telegram -- it replies with a pairing code.

In Claude Code, run:

/telegram:access pair YOUR_CODE

Lock it down:

/telegram:access policy allowlist

Add your group:

/telegram:access group add CHAT_ID --allow USER_ID
